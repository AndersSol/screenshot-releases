Screenshot

A small macOS menubar app for screenshots. Three global hotkeys, a preview, two outcomes — clipboard or file.

Hotkeys

KeyMode
⌘⇧1Full screen
⌘⇧2Window — click the window
⌘⇧3Region — drag a rectangle

In the preview: S saves to Downloads, C copies to clipboard, Esc cancels. ⌘Z undoes.

Editor

Inline tools in the preview window:

  • Highlighter — semi-transparent yellow highlight, additive.
  • Redact — 14 px mosaic redaction. Classic "redacted" look that can't be reconstructed.
  • Crop — drag the rectangle around what you want to keep; everything outside is cropped.

For browser windows (⌘⇧2), the app queries macOS Accessibility for the page content frame and auto-crops the URL bar / tabs / bookmarks. "Show full window" rolls back the crop.

Install

  1. Download Screenshot.dmg below.
  2. Open it, drag Screenshot.app to Applications.
  3. Launch it. Onboarding asks for two macOS permissions:
    • Accessibility for global hotkeys
    • Screen Recording for the actual capture
  4. Click "Grant access" — macOS native prompts walk you straight into System Settings.

Limits
